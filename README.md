# lcplanner-server
## Description  
This is the server repo for lcplanner.

## Installation  
      
clone source code locally   
```$ git clone https://github.com/nancy-sun/lcplanner-server```
   
Install dependencies   
```$ npm install```   
   
Start running   
```$ npm start```
